## 📖 О проекте

**PhotoStudio CRM** — это современная веб-система для управления фотостудией, которая объединяет клиентов, фотографов, администраторов и бухгалтеров в едином пространстве. Система позволяет легко управлять заказами, расписанием, финансами и взаимодействием с клиентами.

### 🛠️ Разработано с использованием

| Инструмент | Назначение |
|------------|------------|
| **Visual Studio 2026** | Разработка бэкенда на ASP.NET Core |
| **Visual Studio Code** | Разработка фронтенда (HTML/CSS/JS) |
| **SQL Server Management Studio 22** | Управление базой данных и выполнение SQL-запросов |

### ✨ Ключевые особенности

| Особенность | Описание |
|-------------|----------|
| 🎨 **Современный дизайн** | Нежно-лавандовая цветовая гамма с плавными анимациями |
| 👥 **Многоролевая система** | 4 роли: Админ, Фотограф, Клиент, Бухгалтер |
| 💳 **Гибкая оплата** | 4 способа оплаты: карта, наличные, онлайн, рассрочка |
| ✨ **Эффекты для фото** | 10+ профессиональных эффектов для обработки снимков |
| 🏛️ **16 тематических залов** | Разноцветные залы для любой фотосессии |
| 📊 **Финансовая аналитика** | Полные отчеты и статистика для бухгалтера |
